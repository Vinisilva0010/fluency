    ║                        ✨ DESTAQUES TÉCNICOS                          ║
    ╠═══════════════════════════════════════════════════════════════════════╣
    ║                                                                       ║
    ║   1. Web Speech API - Síntese de Voz                                 ║
    ║      → utterance.lang = 'en-US'                                      ║
    ║      → utterance.rate = 0.9 (velocidade)                             ║
    ║      → synth.cancel() antes de falar (evita sobreposição)            ║
    ║                                                                       ║
    ║   2. YouTube Embed API                                               ║
    ║      → iframe responsivo (aspect ratio 16:9)                         ║
    ║      → URL dinâmica: youtube.com/embed/{id}                          ║
    ║      → Atributos de permissão configurados                           ║
    ║                                                                       ║
    ║   3. Sistema de Navegação em Árvore                                  ║
    ║      → Estrutura recursiva de passos                                 ║
    ║      → proximo_id define o fluxo                                     ║
    ║      → Suporta loops e múltiplos finais                              ║
    ║                                                                       ║
    ║   4. Prevenção de Farming                                            ║
    ║      → Arrays de rastreamento (completedQuizzes, etc.)               ║
    ║      → Verificação antes de dar pontos                               ║
    ║      → Estado persiste no LocalStorage                               ║
    ║                                                                       ║
    ║   5. Sistema Adaptativo Inteligente                                  ║
    ║      → Cálculo de % em tempo real                                    ║
    ║      → Threshold de 80%                                              ║
